# 阶段工作历史总结：数据可视化制图与论文文献编排

在本次 MathorCup C题（中老年人群高脂血症的风险预警及干预方案优化）的攻坚阶段，我们围绕核心模型的“结果呈现（可视化）”与“学术严谨性（文献编排）”进行了深度的定制和打磨，所有交付物均已同步至仓库的 `main` 分支。

以下为本次对话历史的完整成果总结：

## 一、 核心论文数据可视化制图 (Python & Draw.io)
为了在论文中直观展现模型挖掘出的特征维度、人群异质性与最终的预警决策流，我们通过 Python 制图 (Matplotlib & XML 生成) 制作了以下多维高清图表，并提供对应的开源脚本及可编辑源文件：

1. **组合图表：特征重要度与非线性相关性**
   - **生成文件**：`MathorCup/2.1_特征重要度堆叠图.drawio` / `generate_2_1_stacked_bar.py`
   - **说明**：将随机森林的 Gini 重要度与互信息（MI）结合，通过长条堆叠直观展示各项体检指标（如超敏C反应蛋白、TG等）在总特征重要性中的结构占比。

2. **组合图表：中医体质亚型的发病贡献度对比**
   - **生成文件**：`MathorCup/2.2_体质贡献度对比棒棒糖图.drawio` / `generate_2_2_lollipop.py`
   - **说明**：为了强调“痰湿体质”在所有中医体质中对高脂血症具有绝对的解释力，采用了棒棒糖图（Lollipop Chart）对各体质分（气虚、阴虚等）做降序对比，视觉落差强烈。

3. **高维特征关联：多症状交叉感染的集合交互（Upset Plot）**
   - **生成文件**：`MathorCup/generate_3_1_upset.py`
   - **说明**：针对病患“同时具备高血脂+重度痰湿+活动受限”的复杂交集，克服了传统韦恩图（Venn）在 3 个以上集合时的混乱排版，精确量化多并发症子集的病患人数。

4. **[重点打造] 临床预警决策树流向图 (基于多维风险分层矩阵)**
   - **设计背景**：依据医学界最新的 ASCVD 10年风险评估逻辑，结合本课题“血脂异常-痰湿积分-活动能力受限”三层架构，从零编码了高质预警树和非线性判定矩阵。
   - **(一) 横向宽版原型**：`MathorCup/基于风险分层的预警决策模型全图.drawio` / `generate_flowchart_xml.py`
   - **(二) 纵向学术排版 (纯 PNG)**：`MathorCup/5_1_高质预警决策流程图_Tex专用.png` （适应 LaTeX 单栏或双栏高分辨率呈现，色彩做了柔和护眼处理）/ `generate_4_3_tex_png.py`
   - **(三) 纵向可编辑交互版 (包含原版色直角与柔和色圆角)**：
     - `MathorCup/5_1_高质预警决策流程图_Tex纵向排版.drawio` (柔和色彩，UI圆角切边)
     - `MathorCup/5_1_预警决策流程图_原版色直角纵向排版.drawio` (严厉的红黄绿警戒原款色彩，方正直角框)
     - *技术注法*：自动生成时均开启了 HTML 动态边界换行 (`whiteSpace=wrap;html=1`)，确保在 Draw.io 里手动缩放微调时，长文本不会溢出图形框架。

## 二、 学术严谨性提升：文献编排与 LaTeX 报告融合

为了使《赛题报告》正文（`c_report.tex`）具备坚实的统计学和机器学习积淀，我们为数据预处理、相关性分析与树模型推断精选了 4 篇权威的基石文献，并将其无缝衔接至论文的对应干预与分析环节中：

1. **[1] 赛题原始规则的法定引用**：为 ADL 的加和校验与极值规则设定提供官方的合法声明（避免被批主观臆断）。
2. **[2] Breiman L. 等 (Random Forests, 2001)**：论证了团队在面对五十余个体检医学指标的高度共线性时，为何采用基尼（Gini）重要度构建森林特征筛选，而非简单的线性回归推断。
3. **[3] Hastie T. 等 (The Elements of Statistical Learning, 2009)**：为对数变换（处理偏态TG/尿酸）与 $1\% \sim 99\%$ 分位截尾防极端扰动的核心操作，提供了经典统计学习巨著的范式背书。
4. **[4] Bishop C M. (Pattern Recognition and Machine Learning, 2006)**：在关联性度量模块，以模式识别界著名的 PRML 原理，补充解释了使用“互信息（MI）”来捕捉非线性联动分布的必要性与先进性。
