	\begin{abstract}
			针对中老年人群高血脂症的早筛、分层预警与个体化干预问题，本文基于附件中的 $1000$ 例样本数据，构建了“关键指标筛选--三级风险预警--六个月干预优化”的一体化建模框架。首先，围绕痰湿积分与高血脂确诊标签两个目标，分别计算候选指标与痰湿严重程度的 Spearman 相关系数、互信息，以及随机森林特征重要度，并采用等权综合评分筛选关键指标。结果表明，血脂指标中的 TG、TC、LDL-C、HDL-C 对确诊风险具有最直接的解释力，而在去除直接诊断性血脂指标后，血尿酸、血糖、BMI、活动能力总分及体质积分仍保留较强的预警价值。其次，本文在风险模型训练前加入了数据核验、训练集分位截尾、偏态变量对数变换以及 non-HDL-C、活动受限计数等派生特征构造步骤，并在诊断型模型中将脂质结构变量收口为 non-HDL-C、TG 和 LDL-C 组合，以削弱极端值和结构性冗余带来的扰动。在样例数据上，诊断型模型测试集 AUC 达到 $1.0000$，去除 TC/TG/LDL-C/HDL-C 后的早预警模型 AUC 仍达到 $0.8283$，说明经稳健预处理后，多维度特征融合可实现更稳定的风险判别。结合临床正常值区间、痰湿积分阈值与活动能力约束，本文进一步给出了低、中、高三级风险阈值规则，并提炼出“血脂异常 + 尿酸偏高”、“血脂异常 + 吸烟/饮酒史”、“痰湿积分高 + 活动能力低”等核心高风险特征组合。最后，针对痰湿体质患者，本文建立以干预强度和每周频次为决策变量的六个月优化模型，在年龄约束、活动能力约束及总成本不超过 $2000$ 元的条件下，以最小成本达到既定降分目标。对样本 ID 为 $1,2,3$ 的患者求解得到的最优方案分别为 $(1,10)$、$(1,7)$、$(1,7)$，均能在预算内完成至少 $10\%$ 的痰湿积分下降目标。本文方法兼顾可解释性、可计算性与落地性，可为基层慢病管理中的风险筛查与干预决策提供定量依据。
	\end{abstract}

	\section{问题重述}
	高血脂症是中老年群体中最常见的慢性代谢性疾病之一，既与常规血脂指标直接相关，又与中医体质偏颇、活动能力下降、生活方式不良等因素存在显著耦合。题目给出了九种体质标签及积分、ADL/IADL 活动能力评分、血脂与代谢指标、生活习惯信息以及高血脂确诊标签，要求完成三个层面的任务：
	\begin{enumerate}
		\item 从血常规体检指标和活动能力指标中筛选既能表征痰湿体质严重程度、又能预警高血脂风险的关键指标，并研究九种体质对高血脂风险的贡献差异。
		\item 构建融合多维特征的三级风险预警模型，给出低、中、高风险的阈值划分依据，并识别痰湿体质高风险人群的核心特征组合。
		\item 对体质标签为 $5$ 的痰湿体质患者设计六个月干预方案，在满足年龄、活动能力和预算约束的情况下，以较低成本实现痰湿积分下降目标，并给出样本 ID 为 $1,2,3$ 的最优方案。
	\end{enumerate}

	\section{数据理解与总体思路}
		附件数据共包含 $1000$ 例样本、$37$ 个字段，无缺失值；其中高血脂确诊样本 $793$ 例，未确诊样本 $207$ 例。将主导体质标签按九种体质积分最大项修正后，痰湿体质样本数最多，共 $292$ 例。整体来看，数据具有两个鲜明特征：其一，血脂指标与确诊标签高度同源，若直接用于分类，会得到非常高的判别精度；其二，活动能力、代谢异常与体质积分在排除直接血脂指标后，仍具有提前预警价值。因此本文采用图 \ref{fig:framework} 所示的双层建模思路：
	\begin{enumerate}
		\item 问题一侧重“筛指标”和“看差异”，从统计相关性、信息量和树模型重要度三个角度联合打分；
		\item 问题二将风险识别划分为“诊断型识别”和“早预警识别”，前者追求精度，后者强调前瞻性与可解释阈值；
		\item 问题三把中医调理等级视为随痰湿积分动态变化的基础成本，把活动干预强度和频次作为优化变量，建立约束优化模型。
	\end{enumerate}

		\subsection{数据核验与预处理}
		虽然样例数据没有缺失值和重复样本，但直接“读入即建模”仍然不够稳健。为此，本文在建模前加入了结构核验与稳健预处理两层步骤。

		首先，对原始字段进行一致性检查。结果表明：ADL 五个子题项之和与 ADL 总分完全一致，IADL 五个子题项之和与 IADL 总分完全一致，且 ADL 总分与 IADL 总分之和与活动量表总分完全一致；高血脂二分类标签为 $0$ 时，分型标签均为 $0$，说明标签系统内部协调。但同时发现，原始主导体质标签与九种体质积分最大项并不总是完全一致，样例中共有 $65$ 例样本存在不一致现象。考虑到题目中的“主导体质标签”本应表示九种体质中的主导类型，本文将这 $65$ 例视为标签异常，并统一按九种体质积分最大项对主导体质标签进行修正；其余 $935$ 例样本原标签与积分最大项一致。

				其次，对进入问题二风险模型的连续变量实施稳健化处理。考虑到 TG、血尿酸、BMI 等变量存在偏态与极端值扰动，本文仅在训练集上估计 $1\%$ 与 $99\%$ 分位点，并据此对 HDL-C、LDL-C、TG、TC、空腹血糖、血尿酸和 BMI 进行分位截尾，再将同样的阈值应用于测试集，以避免数据泄漏。设连续变量 $x$ 在训练集上的 $1\%$ 与 $99\%$ 分位点分别为 $q_{0.01}(x)$ 与 $q_{0.99}(x)$，则其稳健化后的截尾变量定义为
			\begin{equation}
				x_i^{(w)}=\min\left\{\max\left(x_i,q_{0.01}(x)\right),q_{0.99}(x)\right\}.
			\end{equation}
			在此基础上，对右偏更明显的 TG 和血尿酸进一步做对数变换：
			\begin{equation}
				\text{TG}_{i}^{(\log)}=\log\left(1+\text{TG}_{i}^{(w)}\right),\qquad
				\text{UA}_{i}^{(\log)}=\log\left(1+\text{UA}_{i}^{(w)}\right).
			\end{equation}
			同时构造 non-HDL-C 这一脂质结构变量：
			\begin{equation}
				\text{nonHDL}_{i}=\text{TC}_{i}^{(w)}-\text{HDL}_{i}^{(w)}.
			\end{equation}
			考虑到 non-HDL-C 与 TC、HDL-C 存在线性确定关系，若再同时引入 TC/HDL-C、LDL-C/HDL-C、TG/HDL-C 等派生比值，会放大同一脂质家族的重复表达。因此，本文在诊断型模型中仅保留 non-HDL-C、TG 和 LDL-C 这一组脂质组合，用 non-HDL-C 概括总胆固醇与保护性胆固醇的净失衡程度，用 TG 表征甘油三酯异常，用 LDL-C 表征低密度脂蛋白负担，从而兼顾医学解释性与变量独立性。
	
			最后，针对活动能力和体质信息加入结构化派生变量。活动变量中，由于 ADL/IADL 子题项与总分之间存在严格加总关系，本文在问题二模型中不再同时放入全部子题项与总分，而是保留 ADL 总分、IADL 总分，并额外构造“活动受限题项数”以补充局部功能损伤信息。设 10 个 ADL/IADL 子题项记为 $a_{ij}$，经验阈值取 $\tau=8$，则第 $i$ 个样本的活动受限计数定义为
			\begin{equation}
				C_i^{(\text{act})}=\sum_{j=1}^{10}\mathbb{I}(a_{ij}<\tau),
			\end{equation}
			其中 $\mathbb{I}(\cdot)$ 为指示函数。该变量反映的是“低分题项有多少个”，因此它补充的是总分无法直接表达的受限范围信息。

			体质变量中，设第 $i$ 个样本的九种体质积分排序后最大值与次大值分别为 $s_{i(1)}$ 和 $s_{i(2)}$，则进一步构造
			\begin{equation}
				S_i^{(\max)}=s_{i(1)},\qquad
				M_i^{(\text{gap})}=s_{i(1)}-s_{i(2)},
			\end{equation}
			分别表示“最强体质倾向的绝对强度”和“第一体质与第二体质之间的优势差距”。由于主导体质标签已在数据核验阶段按积分最大项完成修正，后续建模不再单独构造标签一致性标记。这样处理后，既保留了医学解释性，又降低了冗余表达和极端值对随机森林训练的干扰。需要说明的是，问题一的排序对象仍保持为原始血常规和活动量表指标，上述稳健化步骤主要用于问题二的预测模型与风险分层。

	\section{模型假设}
	\begin{enumerate}
		\item 附件样本已完成临床检验与标签标注，数据可直接用于建模与验证。
		\item 题目给出的临床正常值区间与活动能力约束规则具有统一适用性，视为风险分层与干预求解的硬约束。
		\item 痰湿积分每月变化主要由活动干预强度与频次驱动；中医调理等级在题目中未给出显式降分系数，因此本文将其视为按积分区间自动匹配的基础调理成本。
		\item 在相同活动干预强度下，若每周训练频次低于 $5$ 次，则痰湿积分近似保持稳定；若每周训练频次不低于 $5$ 次，则按题目给出的比例规则递推下降。
		\item 六个月周期内患者的年龄组、性别以及吸烟饮酒史不发生变化，活动量表初值用于确定可选强度上界。
	\end{enumerate}

		\section{问题一：关键指标筛选与体质贡献分析}
			\subsection{双目标筛选思路与评价指标}
				问题一并不是单纯寻找“与痰湿积分最相关”的变量，而是要从血常规体检指标和活动量表评分中筛出一组既能刻画痰湿偏颇程度、又能预警高血脂发病风险的关键指标。因此，本文将其视为一个双目标筛选问题。需要首先说明的是，活动能力变量之间存在明显的结构性共线性：ADL 总分是 5 个 ADL 子题项之和，IADL 总分是 5 个 IADL 子题项之和，而活动量表总分又等于 ADL 总分与 IADL 总分之和。若将总分、整体总分和全部子题项同时放入同一候选池，会导致功能能力维度被重复计分。因此，本文采用分层处理策略：主分析只保留 ADL 总分和 IADL 总分；活动量表总分作为后续问题二风险分层和问题三强度约束的派生变量，不参与问题一主排序。同时，为了检验是否存在某个子题项能够明显优于总分变量，补充分析中再将 10 个 ADL/IADL 子题项与 HDL-C、LDL-C、TG、TC、空腹血糖、血尿酸、BMI 一并放入同一候选池重新排序。基于这一处理，主分析的候选变量集合记为
			\begin{equation}
				\begin{aligned}
					X=\{&\text{ADL总分、IADL总分、HDL-C、LDL-C、TG、TC},\\
					&\text{空腹血糖、血尿酸、BMI}\}.
				\end{aligned}
			\end{equation}
			其中每个候选变量记为 $x_j$，痰湿积分记为 $P$，高血脂二分类标签记为 $y$。对每个候选变量，本文从“单变量相关性”“非线性信息量”和“多变量联合重要度”三个角度进行评价。
		
		首先，为了衡量候选指标对痰湿积分的单调关联程度，计算 Spearman 秩相关系数，并取其绝对值
		\begin{equation}
			R_j=\left|\rho_s(x_j,P)\right|.
		\end{equation}
		这里取绝对值，是因为问题一关心的是“关联强弱”而非“正负方向”。例如 HDL-C 或活动量表总分可能与痰湿积分呈负相关，但这类负向指标同样具有很强的风险识别意义，不能因为方向为负就被排除。
		
		其次，为了避免仅依赖线性或单调关系，进一步计算互信息指标：
		\begin{equation}
			M_j^{(1)}=I(x_j;P),\qquad M_j^{(2)}=I(x_j;y),
		\end{equation}
		其中 $M_j^{(1)}$ 衡量指标对痰湿积分的非线性解释能力，$M_j^{(2)}$ 衡量指标对高血脂标签的非线性预警能力。
		
		再次，为了考察变量之间交互作用下的多变量联合贡献，分别建立随机森林回归模型和随机森林分类模型：
		\begin{equation}
			P=f_{\text{RF-reg}}(X),\qquad y=f_{\text{RF-clf}}(X),
		\end{equation}
		并提取各候选变量在两类模型中的重要度，记为
		\begin{equation}
			F_j^{(1)},\qquad F_j^{(2)}.
		\end{equation}
		
		综上，每个候选指标共得到五类评价量：
		\begin{equation}
			R_j,\quad M_j^{(1)},\quad M_j^{(2)},\quad F_j^{(1)},\quad F_j^{(2)}.
		\end{equation}
		由于这五类统计量量纲不同，且其中有的反映相关强度、有的反映信息量、有的反映树模型重要度，本文不直接对原值加权，而是先分别排序，再进行积分汇总。设共有 $q$ 个候选指标，若第 $j$ 个指标在第 $m$ 类评价下的名次为 $\mathrm{rank}_j^{(m)}$，则定义其名次得分为
		\begin{equation}
			s_j^{(m)}=q-\mathrm{rank}_j^{(m)}+1.
		\end{equation}
		最终综合筛选得分定义为
			\begin{equation}
				S_j=\sum_{m=1}^{5}s_j^{(m)}.
			\end{equation}
			本文主分析共有 $q=9$ 个候选指标，因此在某一评价维度下排名第 $1$、$2$、$9$ 位的指标，分别得到 $9$、$8$、$1$ 分。$S_j$ 越大，说明该指标在五种评价视角下越稳定地靠前，也就越适合同时承担“体质刻画”和“风险预警”两类任务。这种“排序积分”的处理方式比直接对原值做线性加权更稳健，也更符合程序实现。
		
			\subsection{关键指标筛选结果及解释}
				基于上述流程，主分析中 9 个候选指标的综合排序结果依次为 TC、TG、BMI、HDL-C、ADL 总分、LDL-C、空腹血糖、IADL 总分和血尿酸。换言之，本文不是直接“把这些指标列出来”，而是先让每个指标分别在五个维度上独立排序，再看其总积分是否持续靠前。例如，TC 在五类评价中的名次分别为 $2,7,3,4,1$，因此综合得分为
				\begin{equation}
					S_{\text{TC}}=(9-2+1)+(9-7+1)+(9-3+1)+(9-4+1)+(9-1+1)=33.
				\end{equation}
				同理，ADL 总分在五类评价中的名次分别为 $1,2,7,8,9$，因此综合得分为 $23$。这说明 TC、TG 等直接血脂指标在高血脂分类任务上极强，因此排在前列；而 ADL 总分虽然对高血脂标签的单项判别力不如血脂指标，但它在痰湿积分相关性和回归解释能力上持续靠前，因此仍被保留为核心功能指标。

		\subsection{九种体质的风险贡献差异}
		本文采用“描述性风险提升度 + 模型化边际贡献”两层口径刻画九种体质的风险差异。首先，以主导体质标签为分组变量，定义第 $k$ 类体质的风险提升度为
		\begin{equation}
			L_k=\frac{p_k}{p},
		\end{equation}
		其中 $p_k$ 为主导体质标签为 $k$ 的样本确诊率，$p$ 为总体确诊率。若 $L_k>1$，则说明该主导体质组的发病风险高于样本总体平均水平。该指标直观反映了不同主导体质组之间的组间差异，但它只使用了离散标签信息，无法反映同一样本同时具有多种体质偏颇的情形，也未控制年龄、性别、吸烟和饮酒等基础差异，因此更适合作为描述性统计而非最终的“贡献度”定义。

		进一步地，记第 $i$ 个样本第 $k$ 类体质积分为 $x_{ik}$，其标准化变量定义为
		\begin{equation}
			z_{ik}=\frac{x_{ik}-\mu_k}{\sigma_k},
		\end{equation}
		其中 $\mu_k$ 与 $\sigma_k$ 分别为第 $k$ 类体质积分的样本均值和标准差。令 $W_i$ 表示由年龄组、性别、吸烟和饮酒构成的基础控制变量向量，建立标准化逻辑回归模型
		\begin{equation}
			\Pr(Y_i=1)=\frac{1}{1+\exp\left[-\left(\beta_0+\sum_{k=1}^{9}\beta_k z_{ik}+\gamma^\top W_i\right)\right]},
		\end{equation}
			其中 $Y_i$ 表示样本 $i$ 是否患高血脂。此处不再将 TC、TG、LDL-C、HDL-C 等直接诊断变量加入模型，是为了避免强同源血脂指标掩盖体质积分本身的边际作用。需要强调的是，逻辑回归中的系数 $\beta_k$ 作用在对数几率上，而不是直接作用在发病概率上，因此不能把 $\beta_k$ 直接解释为“概率增加了多少”。

			设模型对样本 $i$ 的预测概率为 $\hat{p}_i$，则第 $k$ 类体质积分对该样本发病概率的局部影响为
			\begin{equation}
				\frac{\partial \Pr(Y_i=1)}{\partial z_{ik}}=\hat{p}_i(1-\hat{p}_i)\beta_k.
			\end{equation}
			它表示：在其他变量保持不变时，第 $k$ 类体质积分每增加 $1$ 个标准差，样本 $i$ 的高血脂预测概率会发生多大变化。由于不同样本的基线风险不同，这个变化量并不相同，因此需要对全体样本取平均。于是定义第 $k$ 类体质的平均边际效应为
			\begin{equation}
				C_k=\frac{1}{n}\sum_{i=1}^{n}\hat{p}_i(1-\hat{p}_i)\beta_k.
			\end{equation}
			因此，$C_k$ 可以理解为“第 $k$ 类体质积分每增加 $1$ 个标准差时，高血脂发病概率在全体样本上的平均变化量”。其符号表示影响方向：$C_k>0$ 说明该体质积分升高会提高发病概率，$C_k<0$ 说明其升高与发病概率下降相关；其绝对值越大，说明这种体质对概率的平均扰动越强。进一步定义贡献占比
			\begin{equation}
				G_k=\frac{|C_k|}{\sum_{j=1}^{9}|C_j|},
			\end{equation}
			则 $G_k$ 可以理解为九种体质在模型意义下对总体风险影响强度的相对份额。换言之，AME 先把逻辑回归系数从“对数几率尺度”转换为“概率尺度”，再通过归一化得到可直接比较的贡献占比。

		样例数据表明，两层口径得到的结论并不完全一致。若仅按修正后的主导体质标签分组，特禀质的风险提升度最高，为 $1.047$，血瘀质最低，为 $0.925$，说明九种主导体质在组间发病率上确有高低差异，但整体差距并不悬殊。若转入模型化定义并控制基础信息，则气虚质的平均边际效应最大，贡献占比为 $0.312$；平和质次之，贡献占比为 $0.229$；湿热质和血瘀质的边际效应为负，说明在控制其他条件后，其积分升高与发病概率下降相关。由此可见，主导体质标签的组间确诊率更接近“群体画像”，而标准化逻辑回归的平均边际效应更接近“体质积分本身的边际风险贡献”。在后续风险预警中，本文以后者作为体质贡献差异的主定义，以前者作为辅助描述。

	\subsection{三级风险阈值设计}
	记样本的痰湿积分为 $P$，活动量表总分为 $Q$，血脂异常个数为 $A$，早预警模型输出概率为 $\hat{p}$。综合题目给出的临床正常区间与样例数据中学习到的风险规律，定义三级风险划分规则为
	\begin{equation}
		R=
		\begin{cases}
			\text{高风险}, & (A\ge 1 \land P\ge 60)\ \lor\ (A=0 \land P\ge 80 \land Q<40)\ \lor\ \hat{p}\ge 0.80,\\
			\text{中风险}, & A\ge 1\ \lor\ P\ge 60\ \lor\ (\hat{p}\ge 0.45 \land Q<60),\\
			\text{低风险}, & \text{其余情形}.
		\end{cases}
		\label{eq:risk_rule}
	\end{equation}
	其中，$A$ 的计算依据为 TC、TG、LDL-C 是否超过题目给定上限，以及 HDL-C 是否低于题目给定下限。该规则兼顾了三方面信息：第一，临床血脂异常是最直接的危险信号；第二，痰湿积分高而活动能力低的人群，即便暂未表现出明显血脂异常，也可能处于早期高风险状态；第三，模型输出概率用来兜底吸收多变量之间的非线性交互关系。

			低风险 & 血脂指标全部正常，痰湿积分 $<60$，活动量表总分较高，且模型输出概率较低\\
			中风险 & 至少存在一项血脂异常，或痰湿积分 $\ge 60$，或模型概率达到中等水平且活动能力不足\\
			高风险 & 满足“血脂异常 + 痰湿积分高”、“血脂正常但痰湿积分极高且活动能力很低”或模型概率 $\ge 0.80$ 的任一情形\\
			\bottomrule
		\end{tabularx}
	\end{table}

	按式(\ref{eq:risk_rule}) 对样例数据分层后，低、中、高风险样本数分别为 $59$、$292$、$649$；在修正后主导体质标签为痰湿质的 $292$ 例样本中，高风险人数为 $207$，占比达到 $70.89\%$。

	\subsection{高风险核心特征组合识别}
	对高风险样本进行特征组合统计后可见，出现频次最高的模式主要包括：
	\begin{enumerate}
		\item 血脂异常 + 尿酸偏高；
		\item 血脂异常 + 吸烟史/饮酒史；
		\item 血脂异常 + 尿酸偏高 + 吸烟史 + 饮酒史；
		\item 痰湿积分高 + 低活动量 + 血脂异常；
		\item 血脂异常 + BMI 偏高 + 尿酸偏高。
	\end{enumerate}
	其中，“血脂异常 + 尿酸偏高”是最高频的高风险组合；若再叠加吸烟、饮酒、BMI 偏高或活动能力低，则风险进一步放大。这一结果说明高血脂并非孤立的单指标异常，而是代谢异常、生活方式风险与体质偏颇共同驱动的结果。对痰湿体质人群而言，痰湿积分本身提供了中医维度的先验风险，而活动能力下降则意味着代谢调节能力和后续干预耐受度均相对较差，因此应优先纳入高风险监测名单。

	\section{问题三：痰湿体质患者的六个月干预优化}
	\subsection{决策变量与约束条件}
	对主导体质标签为 $5$ 的患者，设活动干预强度为 $u$，每周训练频次为 $f$。根据题目要求：
	\begin{enumerate}
		\item $u$ 受年龄组和活动量表总分的双重约束；
		\item $f\in\{1,2,\dots,10\}$；
		\item 六个月总成本不超过 $2000$ 元；
		\item 痰湿积分目标设为六个月后至少下降 $10\%$，即 $s_6\le 0.9 s_0$。
	\end{enumerate}
	活动干预的月度降分比例定义为
	\begin{equation}
		r(u,f)=
		\begin{cases}
			0, & f<5,\\
			0.03(u-1)+0.01(f-5), & f\ge 5.
		\end{cases}
	\end{equation}
	于是痰湿积分的月度递推关系为
	\begin{equation}
		s_{m+1}=s_m\bigl(1-r(u,f)\bigr),\qquad m=0,1,\dots,5.
	\end{equation}

	中医调理等级根据当月积分自动匹配：$0$--$58$ 分为一级、$59$--$61$ 分为二级、$\ge 62$ 分为三级。记对应月成本函数为 $c_{\text{tcm}}(s_m)$；活动训练的单次成本为 $c_u$，则六个月总成本为
	\begin{equation}
		C=\sum_{m=1}^{6}\left[c_{\text{tcm}}(s_{m-1})+4f\,c_u\right].
	\end{equation}
	据此建立如下优化模型：
	\begin{equation}
		\min_{u,f}\ C
	\end{equation}
	满足
	\begin{equation}
		s_6\le 0.9s_0,\qquad C\le 2000,\qquad u\in\mathcal{I}(\text{年龄组},Q),\qquad f\in\{1,\dots,10\},
	\end{equation}
	其中 $\mathcal{I}(\text{年龄组},Q)$ 表示由年龄组和活动量表总分共同决定的允许强度集合。

	\subsection{样本 1、2、3 的最优方案}

	\subsection{患者特征与最优方案的匹配规律}
	综合所有可行方案的搜索结果，可以总结出如下匹配规律：
	\begin{enumerate}
		\item 若年龄较大或活动量表总分偏低，则强度选择空间被压缩，此时应优先通过增加频次改善干预效果；
		\item 若初始痰湿积分处于 $59$--$62$ 分附近，则前两个月的积分下降会降低中医调理等级，从而减少后续调理成本，因此适度提高前期训练频次往往更划算；
		\item 若患者活动能力较高但目标降幅不大，盲目追求高强度并不一定优于低强度中等频次，因为训练成本上涨快于收益；
		\item 预算约束下，“低强度 + 合理频次 + 动态降低调理等级”是最稳健的经济策略。
	\end{enumerate}

	\section{模型检验与结果讨论}
	本文模型的检验主要从三个角度展开。
	\begin{enumerate}
			\item {\heiti 结果一致性检验}：诊断型模型的重要特征集中在 TG、non-HDL-C、LDL-C 及少量代谢变量，与高血脂临床诊断中“甘油三酯异常 + 非 HDL 胆固醇负担 + LDL 负担”的识别逻辑一致；早预警模型的重要特征集中在血尿酸、血糖、BMI 和体质、活动变量，与慢病早筛的医学常识相符。
		\item {\heiti 阈值可解释性检验}：三级风险阈值同时引用了临床正常值区间、痰湿积分与活动能力规则，不依赖黑盒模型单独输出，便于基层使用者理解与执行。
		\item {\heiti 干预可行性检验}：样本 $1,2,3$ 的最优方案均满足年龄约束、活动能力约束和预算约束，且最终痰湿积分均达到既定目标，说明优化模型具有实际可操作性。
	\end{enumerate}
	需要指出的是，样例数据中确诊样本占比达到 $79.3\%$，类别分布并不均衡，这会抬高部分单体质组的表观确诊率。因此，若后续补充了更多社区筛查数据，应优先采用分层抽样、代价敏感学习或重采样手段增强模型稳健性。

	\section{模型评价与改进方向}
	\subsection{模型优点}
	\begin{enumerate}
		\item 将“诊断识别”和“未病预警”分开建模，避免直接血脂指标掩盖其他变量的预警价值；
		\item 风险等级规则可解释性强，便于在社区卫生服务场景中落地；
		\item 干预模型直接嵌入了题目给定的强度、频次、年龄与预算约束，输出方案具有较强操作性；
		\item 建模流程统一，问题一的筛选结果可自然过渡到问题二和问题三。
	\end{enumerate}

	\subsection{模型不足}
	\begin{enumerate}
		\item 中医调理等级只给出了成本标准，未给出独立的降分函数，因此本文将其作为基础成本而非单独的收益项；
		\item 活动干预效果采用固定月度下降比例，尚未考虑患者依从性波动和边际收益递减；
		\item 样例数据存在类别不平衡现象，且不同体质之间可能存在多重共线性，使单体质贡献的解释受到一定干扰。
	\end{enumerate}

	\subsection{改进方向}
	\begin{enumerate}
		\item 若能补充随访数据，可将问题三拓展为动态规划或模型预测控制问题；
		\item 若能获得更细致的中医调理效果数据，可把饮食调理、穴位按摩、八段锦和代茶饮拆分为独立决策变量；
		\item 若后续加入更多地区、更多时间段样本，可建立跨区域迁移学习模型，提高外部泛化能力。
	\end{enumerate}
